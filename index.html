  • 7 types of HTML5 content

This is our first html tags:

  • <html> - open-closed tag in start of the html page after doctype
  • <head> - open-closed tag in start of the html page after html tag
  • <meta> - self-closed tag in head of the html page after for set charset
  • <body> - open-closed tag in start of content our web page after </head> tag

HTML5 concepts

  • Block-Level Elements
    • Render to begin on a new line
    • May contain inline or other block-level elements
    • Roughli Flow Content(HTML5 category)
  • Inline Elements
    • Render on the same line by default
    • May only contain other inline elements
    • Roughly Phrasing Content(HTML5 category)
  • HTML5 replaces these definitions with more complex set of content categories.
  • However, this distinction remains practical because it aligns well with exiting css rules.

Available 7 types of content

They are:
  • Metadata content
  • Flow content
  • Sectioning content
  • Heading content
  • Phrathing content
  • Embedded content
  • Interactive content

Using spesial symbols in html,or Entity references:

For using tag "<" - we should use "&lt;".
For using tag ">" - we should use "&gt;".
For using tag "&" - we should use "&amp;".
For using simbol "©" - we should use "&copy;".
For make not braking spase we should use "&nbsp;".
For email when we use qoutes, we should use "&quot;".
