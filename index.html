  • "<"body">" - open-closed tag in start of content our web page after "" tag

HTML5 concepts

  • Block-Level Elements
    • Render to begin on a new line
    • May contain inline or other block-level elements
    • Roughli Flow Content(HTML5 category)
  • Inline Elements
    • Render on the same line by default
    • May only contain other inline elements
    • Roughly Phrasing Content(HTML5 category)
  • HTML5 replaces these definitions with more complex set of content categories.
  • However, this distinction remains practical because it aligns well with exiting css rules.

  • Metadata content
  • Flow content
  • Sectioning content
  • Heading content
  • Phrathing content
  • Embedded content
  • Interactive content
